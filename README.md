# Plant Disease Detection using Deep Learning

This project detects plant diseases from leaf images using
Convolutional Neural Networks (CNN).

## Technologies Used
Python
TensorFlow
OpenCV
NumPy

## Dataset
Plant leaf images dataset (Tomato diseases)

Classes:
- Tomato Early Blight
- Tomato Late Blight
- Tomato Healthy

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Train the model
python train_model.py

3. Predict disease
python predict.py

## Output
The model predicts the disease from a leaf image.# plant-disease-detection
